# SQL
<hr>
요즘 사용하는 대부분의 데이터베이스 시스템은 관계형 모델을 바탕으로 설계되어 있습 니다. 
관계형 데이터 베이스 시스템을 사용하기 위해서는 이를 명령하고 처리할 수 있는 별도의 SQL 명령을 학습해야 합니다.  

* [SQL](./03)
* [권한 설정](./29)

SQL은 데이터베이스의 구조와 데이터를 처리 합니다. SQL 쿼리는 크게 2종류로 구분을 할 수 있습니다.

* DDL
* DML

<br>
