19 UNION

지금까지의 데이터 작업들은 모두 1개의 테이블의 기준으로 작업했습니다. 관계형 데이 터베이스의 특징은 데이터를 다수의 테이블에 분산하여 저장한다는 것입니다. 또한 분산 되어 있는 자료들은 관계를 맺고 데이터를 검색 처리합니다.

MySQL에서는 여러 개의 테이블을 결합하여 처리할 수 있는 UNION과 JOIN 명령 키워 드를 지원합니다.
