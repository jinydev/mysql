02.6.파일 삽입

백업과 복원은 mysql의 외부 유틸리티 프로그램을 통하여 실행됩니다. 하지만 별도의 SQL 명령을 통하여 외부 파일을 읽어올 수 있습니다. 읽은 파일을 통하여 테이블에 데이 터를 삽입합니다.

| 쿼리 문법 |

LOAD DATA INFILE 파일명 INTO TABLE 테이블명 옵션;
