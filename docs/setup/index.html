관리툴

MySQL은 폭넓은 사용과 인기에 더불어 기존에 불편한 콜솔 작업창 이외에 다양한 GUI 관리 프로그램이 있습니다.

GUI 관리 프로그램을 이용하여 데이터베이스의 기초 작업 및 데이터 수정을 보다 쉽게 처리할 수 있습니다. 번 장에서는 자주 사용하는 관리자 툴에 대해서 간략하게 소개합 니다. 보다 자세한 것은 관련 사이트에서 확인 가능합니다.
