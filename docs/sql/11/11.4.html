11.4.* 와일드 카드

아스테리스크(*) 와일드 카드는 앞의 전체 컬럼을 출력할 때 한 번 사용했습니다. * 기호 는 전체를 대신하는 기호입니다. SELECT 컬럼명을 나열하는 * 와일드 카드를 적용하면 전체의 컬럼 지정을 의미합니다.

| 쿼리 문법 |

SELECT * FROM 테이블명;

| 예제 쿼리 |

SELECT * FROM members;

다음 select 쿼리는 회원 테이블에서 전체의 컬럼 필드의 정보를 가지고 옵니다.
