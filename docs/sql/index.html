SQL

요즘 사용하는 대부분의 데이터베이스 시스템은 관계형 모델을 바탕으로 설계되어 있습 니다. 관계형 데이터 베이스 시스템을 사용하기 위해서는 이를 명령하고 처리할 수 있는 별도의 SQL 명령을 학습해야 합니다.

  • SQL
  • 권한 설정

SQL은 데이터베이스의 구조와 데이터를 처리 합니다. SQL 쿼리는 크게 2종류로 구분을 할 수 있습니다.

  • DDL
  • DML


데이터 정의 쿼리(Data definition language, DDL)

DDL는 데이터를 저장하는 구조를 처리하는 쿼리를 말합니다.

  • 데이터베이스
  • 테이블
  • 데이터 타입
  • 테이블 수정
  • 속성

데이터 조작 쿼리 (Data manipulation language, DML)

DML은 저장된 데이터를 가공하는 쿼리를 말합니다.

  • 데이터 삽입
  • 데이터 조회
  • 조건
  • 데이터 제한
  • 데이터의 정렬
  • 복사
  • 별칭
  • 데이터 수정
  • 데이터 삭제
  • UNION
  • 조인
  • 뷰(VIEW)
  • 프로시저
  • 트리거
  • 트랜잭션
