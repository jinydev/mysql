14 데이터의 정렬

테이블에 입력된 데이터의 출력은 입력한 데이터 순서대로 출력됩니다. 데이터를 검색하 여 출력할 때 무작위로 데이터가 출력되면 결과를 쉽게 분석하기 어렵습니다.

데이터를 검색하고 출력할 때는 처리한 결과를 쉽게 알기 위해서 정렬하여 출력합니다. 정렬 기능은 결과물을 보기 좋게, 가독성 있게 만드는 중요한 기능입니다.
