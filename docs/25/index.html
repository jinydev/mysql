25 함수

MySQL은 쿼리 명령 이외에 데이터를 처리할 수 있는 다양한 내장 함수를 제공하고 있습 니다.

내장 함수들은 기본적으로 제공되는 함수들로 간단한 연산, 시스템 정보, 데이터 가 공 작업을 할 수 있습니다.

MySQL 5.0 이상 버전에는 사용자가 직접 함수를 생성하고 사용할 수 있는 기능을 제공 합니다.
