# 15 복사 
기존 테이블 데이터를 기반으로 새로운 테이블과 데이터를 삽입할 수 있습니다. 
테이블을 생성할 때의 CREATE TABLE 명령과 데이터를 출력하는 SELECT 명령을 함께 결합하여 테이블을 복사할 수 있습니다.  

테이블 및 데이터 복사는 기존에 존재하는 데이터를 기반으로 작업을 하기 때문에 빠르게 처리할 수 있습니다.
